
# Best Wishes
Mobile application which create greeting cards from chosen picture and wishes.
- Birthday cards
- Anniversary cards
- Load your own photo



## Technologies
- Kotlin
- Android
- XML
- Retrofit
- Navigation component
- Coroutines
- Espresso
