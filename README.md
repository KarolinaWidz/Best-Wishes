# Best Wishes
Mobile application which creates greeting cards from chosen pictures and wishes.
- Birthday cards
- Anniversary cards
- Load your own photo

![design](./app_photos/Best%20wishes%20design.png)

## Technologies

- Kotlin
- Android
- XML
- Retrofit
- Coroutines
- Espresso
- Moshi
- Hilt
