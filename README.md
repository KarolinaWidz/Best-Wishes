
# Best Wishes
Mobile application which creates greeting cards from chosen pictures and wishes.
- Birthday cards
- Anniversary cards
- Load your own photo



## Technologies
- Kotlin
- Android
- XML
- Retrofit
- Navigation component
- Coroutines
- Espresso
